# Reporting Platform Acceptance Checklist

## 1. Report Builder & Templates
- [ ] Drag-and-drop designer correctly places dimensions and metrics.
- [ ] Calculated fields and conditional formatting functional.
- [ ] Academic, Operations, and AI templates successfully deploy.

## 2. Ad-hoc Analytics
- [ ] Semantic Models translate successfully into interactive SQL queries.
- [ ] Saved queries and reusable filters can be applied to new reports.

## 3. Scheduling & Distribution
- [ ] CRON engine accurately triggers daily, weekly, and custom schedules.
- [ ] Role-Based Distribution correctly applies RLS to generated exports.
- [ ] In-app Delivery and Download Center retain files securely.

## 4. Exports & APIs
- [ ] PDF generation handles print layouts without truncation.
- [ ] Excel/CSV exports accurately reflect grouping and sorting.
- [ ] Analytical APIs for scheduling and building are authenticated and active.

## 5. Security & Auditing
- [ ] Draft and Version History function correctly.
- [ ] Approval workflow prevents unverified reports from broad distribution.
- [ ] Audit logs track `Report Exported` and `Schedule Updated`.
- [ ] Usage Analytics correctly identify `Most Used Reports`.
