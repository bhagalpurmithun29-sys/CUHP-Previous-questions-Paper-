# Super Administrator Manual

## Managing the Platform
As a `super_admin`, you have access to the central orchestration systems.

### 1. Platform Operations Center (`/admin/ai-admin`)
- Control overarching routing rules.
- Enable or disable AI model providers instantly.
- Monitor active alert queues (e.g., "High Latency" warnings).

### 2. Platform Orchestration Dashboard (`/admin/platform`)
- View the unified map of dependencies between RAG, Prompts, and the Gateway.
- Manage Feature Flags (e.g., toggling the Study Planner module).
- Run the Pre-Production Checklist via "Run Full Validation".

### 3. Prompt Management (`/admin/prompts`)
- Lock and version core AI instructions.
- Test variations without affecting the live platform.

### 4. Safety & Feedback (`/admin/safety`, `/admin/feedback`)
- Oversee the human-in-the-loop review queues.
- Accept or reject AI improvements suggested by reviewers.
